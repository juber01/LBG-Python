# THIS IS OUR NEW README FILE

## Initialise a new repo
```
git init
```

## To stage all files
```
git add *
```

## To commit staged files
```
git commmit -m "meaningful message"

## To push commits to remote repo
```
git push -u REMOTE_NAME BRANCH_NAME
```

## To create a branch
```
git branch NEW_BRANCH_NAE
```
